# Koch Rajbongshi Development Council Website

Official website for Koch Rajbongshi Development Council, an educational institution serving the Koch Rajbongshi community of Assam.

## Overview

A plain HTML website built with Tailwind CSS v4 (CDN) featuring:
- Homepage with institution overview
- About Us page
- APSC Application page
- Skill Development Application page

## Technology Stack

- **HTML5**: Plain HTML pages
- **Tailwind CSS v4**: Via CDN (no build process required)
- **JavaScript**: Minimal vanilla JS for interactivity
- **Responsive Design**: Mobile-first approach

## Color Palette

### Olive Leaf
- Primary colors ranging from #f6fee6 (olive-50) to #172301 (olive-950)
- Used for headers, navigation, primary buttons

### Golden Glow
- Accent colors ranging from #fffde6 (golden-50) to #242100 (golden-950)
- Used for highlights, secondary buttons, call-to-actions

## Getting Started

Simply open `index.html` in a web browser. No build process or dependencies required.

## Navigation

- All navigation links are always visible
- Mobile: Stacked vertically
- Desktop: Horizontal layout
- No hamburger menu for better accessibility
